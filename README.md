# Sam's Jams — web

Static HTML exported from the Google Stitch project **Home - Sam's Jams** (“The Neon Pulse” design system). Uses Tailwind via CDN, Google Fonts, and Material Symbols.

## Pages

| File | Description |
|------|-------------|
| [index.html](index.html) | Home — dark theme |
| [home-light.html](home-light.html) | Home — light theme |
| [top100.html](top100.html) | Top 100 — dark theme |
| [top100-light.html](top100-light.html) | Top 100 — light theme |

Primary navigation links between Home and Top 100 (dark ↔ dark, light ↔ light).

## Local preview

Open `index.html` in a browser, or from this folder run:

```bash
npx serve .
```
